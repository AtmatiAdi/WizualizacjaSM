Detailed Description

The Micromouse thread class. This class communicates with MainWindow by signals, can send move and rotation command to the robot. Is used for controlling the robot in Maze class instance.

◆ run()

void Micromouse::run ( )
override

main thread loop.

Algorithm gets current distance and rotation to next path cell and simple sends command to the robot from Maze class instance. This function continuously check if IsRunning = true to execute algorithm. If you set IsRunning to false this function will execute to end and skip loop to finish thread, all informations are saved and you can start micromouse thread again. For better results of reaching to the target, improve this function.
