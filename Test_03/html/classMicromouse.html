void run () override
 main thread loop. Simple algorithm gets current distance and rotation to next path cell and simple sends command to the robot. This function continuously check if IsRunning is true to execute algorithm. If you set IsRunning to false this function will break the loop, all informations are saved and you can start micromouse thread again. For better results of reaching to the target, improve this function.
