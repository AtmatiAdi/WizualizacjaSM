{\footnotesize\ttfamily void Micromouse\+::run (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [override]}}



main thread loop. 

Algorithm gets current distance and rotation to next path cell and simple sends command to the robot from \mbox{\hyperlink{classMaze}{Maze}} class instance. This function continuously check if \mbox{\hyperlink{classMicromouse_a2b061bd28fb66cb6ddc0e5956f7cf6e3}{Is\+Running}} = true to execute algorithm. If you set Is\+Running to false this function will execute to end and skip loop to finish thread, all informations are saved and you can start micromouse thread again.
